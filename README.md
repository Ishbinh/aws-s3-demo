# aws-s3-demo
AWS S3 hands-on demo including static website hosting and access control

This repository contains hands-on AWS S3 demos performed while learning AWS cloud.

## Projects included

### 1. Static Website Hosting using S3
- Created S3 bucket
- Enabled static website hosting
- Added bucket policy
- Hosted HTML website

### 2. S3 Access Control Demo
- Created private bucket
- Enabled block public access
- Tested restricted access
- Verified root user permissions

## Services used

- AWS S3
- IAM
- Bucket Policy
- Static Website Hosting

## Purpose

This repository is created as proof of hands-on practice while learning AWS.
